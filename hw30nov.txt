1. Какой тип хранения данных использует GIT. В чем его преимущества?

В Git единицей хранения данных является объект (англ. object), который однозначно определяется 40-символьным
 хешем sha1. В объектах Git хранит почти всё: коммиты, содержимое файлов, их иерархию. Сначала объекты 
представляют из себя обычные файлы в папке .git/objects, а после git gc упаковываются в .pack-файлы.
 Для экономии дискового пространства содержимое всех объектов дополнительно сжимается с помощью zlib.
Также хранение объектов целиком позволяет делать надежное слияние веток с разрешением конфликтов.
